# Laboratorium 7
Projekt po wykonaniu ćwiczenia laboratoryjne nr 7.
* wykonanie inżynierii wstecznej - baza `movies.db`,
* wygenerowanie kontrolera na podstawie modelu `Movie` i kontekstu `MovieContext`
* stworzenie prostego kontrolera API, który zwraca przefiltrowane wytwórnie filmowe
* modyfikacja formularza dodawania filmu, który odwołuje się do API i wypełnia element typu select z nazwami wytwórni
* modyfikacja metody akcji tworzenia nowego filmu, aby oprócz dodania filmu dodawany był rekord do tabeli asocjacyjnej filmu z wytwórniami
* modyfikacja konfiguracji kontekstu - zdefiniowanie klucza głównego w tabeli asocjacyjnej film - wytwórnia
